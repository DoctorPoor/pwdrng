# pwdrng

Simple CLI to generate passwords.

## Prerequisite

- Install [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
- Install [Go](https://go.dev/doc/install)

## Installation

```
$ git clone https://github.com/doctorpoor/pwdrng && cd pwdrng
```

## Usage

```
$ go run pwdrng.go
Generated password: 
8ly3180>j<O)3LpW
```
